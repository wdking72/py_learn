# 文件读写笔记

## 1. 写入文件

```javascript
// JS：fs 模块
const fs = require('fs');
fs.writeFileSync('test.txt', '你好世界');
```

```python
# Python：open + with 语句
with open('test.txt', 'w') as f:
    f.write('你好世界')
```

**关键点**：
- `with` 语句自动关闭文件，不需要手动 `f.close()`
- `'w'` 是写入模式，文件存在会覆盖

---

## 2. 读取文件

```javascript
// JS
const content = fs.readFileSync('test.txt', 'utf8');
```

```python
# Python：三种方式
with open('test.txt', 'r') as f:
    content = f.read()        # 方式1：读取全部内容为字符串

with open('test.txt', 'r') as f:
    lines = f.readlines()     # 方式2：按行读取，返回列表

with open('test.txt', 'r') as f:
    for line in f:            # 方式3：逐行遍历
        print(line.strip())   # strip() 去掉末尾换行符
```

**区别**：Python 用 `with` 管理文件生命周期，JS 需要手动处理。

---

## 3. 文件模式

| 模式 | 说明 | 文件不存在时 |
|---|---|---|
| `'r'` | 读取（默认） | 报错 FileNotFoundError |
| `'w'` | 写入（覆盖） | 创建新文件 |
| `'a'` | 追加（末尾添加） | 创建新文件 |
| `'r+'` | 读写 | 报错 |

```python
# 追加写入
with open('test.txt', 'a') as f:
    f.write('\n追加的内容')
```

---

## 4. 安全读取（处理文件不存在）

```javascript
// JS：先判断再读
if (fs.existsSync('test.txt')) {
    const content = fs.readFileSync('test.txt', 'utf8');
}
```

```python
# Python：try/except 更 Pythonic
try:
    with open('test.txt', 'r') as f:
        content = f.read()
except FileNotFoundError:
    print("文件不存在")
    content = ""
```

**推荐**：用 `try/except` 而不是 `if os.path.exists()`，避免判断和读取之间文件被删除的竞态问题。

---

## 5. 速查表

| 操作 | JavaScript | Python |
|---|---|---|
| 写入 | `fs.writeFileSync(f, data)` | `with open(f, 'w') as f: f.write(data)` |
| 读取 | `fs.readFileSync(f, 'utf8')` | `with open(f, 'r') as f: f.read()` |
| 追加 | `fs.appendFileSync(f, data)` | `with open(f, 'a') as f: f.write(data)` |
| 判断存在 | `fs.existsSync(f)` | `os.path.exists(f)` |
| 逐行读取 | `fs.readFileSync().split('\n')` | `for line in f:` |

## 6. 注意事项

```python
# ❌ 捕获太宽泛
except:                    # 会吞掉所有错误

# ✅ 捕获具体错误
except FileNotFoundError:  # 只处理文件不存在
```
