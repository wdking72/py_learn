# Python 进阶学习计划

## 背景

- 已完成 Python 基础（17个概念，6个阶段）
- 用户是 JS 开发者，有 Koa/Next.js 后端经验，用过 MongoDB
- 目标：FastAPI 后端 + Agent 开发，生产级别

## 学习路线

| 阶段 | 标题 | 目标 | 预估 |
|---|---|---|---|
| 1 | Python 后端基础 | 从 JS 后端过渡到 Python 后端思维 | 1-2 次 |
| 2 | FastAPI 入门 | 能搭建基本的 API 服务 | 2-3 次 |
| 3 | PostgreSQL + SQLAlchemy | 能连接和操作 PostgreSQL | 2-3 次 |
| 4 | 认证与中间件 | 实现用户认证和权限控制 | 2 次 |
| 5 | 生产级进阶 | 能写出生产级别的代码 | 2-3 次 |
| 6 | Agent 开发基础 | 理解 LLM 调用和 Agent 原理 | 2 次 |
| 7 | Agent 框架 | 能用框架搭建 Agent | 3 次 |
| 8 | RAG + pgvector | 能做生产级的 RAG 应用 | 3 次 |

**总计**：约 18-22 次学习

## 数据库选型

- PostgreSQL + pgvector
- 一个数据库搞定关系数据和向量存储
- 适合 Agent 开发的 RAG 场景
